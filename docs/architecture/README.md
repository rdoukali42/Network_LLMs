# Architecture Documentation

This directory contains system architecture diagrams and workflow visualizations.

## Available Diagrams

### Workflow Diagrams
- **Complete_Technical_Workflow.drawio**: Full system workflow with all components
- **Multi_Agent_Workflow_Technical.drawio**: Detailed multi-agent interaction flow
- **Agents_Only_Workflow.drawio**: Simplified agent-focused workflow
- **Simple_Agents_Tools_Workflow.drawio**: Agent and tool interaction patterns
- **Simple_Agents_Workflow.drawio**: Basic agent workflow
- **Process.drawio**: High-level process flow

## Viewing Diagrams

These diagrams are in Draw.io format (.drawio). You can view and edit them using:

1. **Online**: [diagrams.net](https://app.diagrams.net/)
2. **VS Code**: Install the "Draw.io Integration" extension
3. **Desktop**: Download [draw.io desktop app](https://github.com/jgraph/drawio-desktop/releases)

## Usage

To open a diagram:
1. Visit https://app.diagrams.net/
2. Select "Open Existing Diagram"
3. Upload the .drawio file from this directory

## Contributing

When updating diagrams:
- Keep diagrams consistent with actual implementation
- Use clear, professional styling
- Export PNG versions for README if needed
- Document major changes in commit messages
